# github-issue-printer

Print your repo issues by issue numbers with format

- input:
  - issue number list.
  - mixing with space, '#', '\n' or other non-numeric characters is ok.
```
{num1} {num2}
#{num3} #{num4}
```

- output
```
#{num1} {title1}
#{num2} {title2}
#{num3} {title3}
#{num4} {title4}
```

[Use online](https://vincentngthu.github.io/github-issue-printer/)
